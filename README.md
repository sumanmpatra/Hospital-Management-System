## Hospital Management System (Java Swing Desktop Application)

- A full-featured desktop application built using Java, Swing, and AWT to streamline daily hospital operations.
- The system includes a secure login interface followed by an intuitive reception dashboard that manages core functionalities such as patient registration, room allocation, doctor and employee information, appointment handling, and ambulance coordination.
- Designed with a modular structure, the project focuses on efficient data flow, simplified UI navigation, and real-time management of hospital resources, providing a complete administrative toolkit for healthcare environments.


## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies
